# 调用助手

LangGraph Studio 允许您测试图的不同配置和输入。它还提供了执行过程中图的直观可视化，便于查看哪些节点正在运行以及每个节点的输出是什么。

1. LangGraph Studio 用户界面显示所选助手的可视化。
    1. 在左侧窗格的左上角下拉菜单中，选择一个助手。
    1. 在左侧窗格的底部，编辑 `输入` 并 `配置` 助手。
    1. 选择 `提交` 以调用所选助手。
1. 在右侧窗格中查看调用的输出。
